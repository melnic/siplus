# Links para funionalidades:
**Design Simpligicado**  
Torna a interface mais enxuta, com mais espaços para dados.  
[Design Simplificado](https://github.com/melnic/siplus/raw/refs/heads/master/redesign.user.js)

**Scan de Conflitos**  
Exibe na agenda semanas as ações que estão em conflito de espaço ou intervelo entre ações menor que 1 hora (dificultando desmontagens e montagens).  
[Scan de conflitos](https://github.com/melnic/siplus/raw/master/scanconflitos.user.js)

**Gerador de Carta Proposta**  
Extrai os dados da ação e implementa menu de geração de carta.  
[Gerador de carta proposta](https://github.com/melnic/siplus/raw/master/carta_proposta.user.js)

**Revisor de ações**  
Verifica se ações estão inseridas de modo adequado. Precisa ter interface melhorada.  
[Revisor de ações](https://github.com/melnic/siplus/raw/master/revisor.user.js)

**Aprovação Simples Estatístico**  
Remove a janela de confirmação de aprovação, agilizando trabalho.  
[Aprovar Estatístico](https://github.com/melnic/siplus/raw/refs/heads/master/aprovar_estatistico.user.js)

**Destacador de Feriados**  
Destaca nos calendários semanal e mensal os dias em que a unidade está aberta ou fechada.  
[Destacador de feriados](https://github.com/melnic/siplus/raw/refs/heads/master/feriados.user.js)
